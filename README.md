Zadanie 1 (exercise 1)
-
Program oblicza n-ty wyraz ciągu Fibonacciego trzema metodami (iteracyjnie, macierzowo, z gmpy2) i zapisuje wyniki do plików tekstowych.

The program calculates the n-th Fibonacci number using three methods (iterative, matrix method, and gmpy2) and saves the results to text files.

Zadanie 2 (exercise 2)
-
Program przetwarza dane o liniach tramwajowych w Krakowie, tworzy statystyki przystanków i generuje interaktywną mapę folium z trasami i przystankami.

The program processes data on tram lines in ćracow, generates stop statistics, and creates an interactive folium map with routes and stops.

Zadanie 3 (exercise 3)
-
Program pobiera dane giełdowe z Yahoo Finance i pandas, wyznacza daty przecięć średnich kroczących (golden cross) i maksymalną dzienną liczbę sprzedanych BTC.

The program fetches stock data from Yahoo Finance using pandas, identifies dates of moving average crossovers (golden cross), and calculates the maximum daily number of BTC traded.
